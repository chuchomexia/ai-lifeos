# Skills Index

Guía para el asistente sobre cuándo y cómo usar cada skill.

## Estructura de Skills

Cada skill representa un área de vida basada en el método *Wheel of Life* (por definir).

## Áreas Candidatas

- [ ] Salud física
- [ ] Salud mental / Bienestar emocional
- [ ] Relaciones / Pareja
- [ ] Carrera / Trabajo
- [ ] Finanzas
- [ ] Desarrollo personal / Aprendizaje
- [ ] Diversión / Recreación
- [ ] Entorno físico / Hogar
- [ ] (Por validar con Jesús)

## Cómo Usar Este Índice

1. Cada skill tendrá su propio archivo `.md` en esta carpeta
2. Este índice documenta cuándo activar cada skill
3. El system prompt referenciará este índice para orquestación

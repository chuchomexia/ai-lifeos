# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

LifeOS is a personal operating system for Jesús - an AI assistant designed to function as an honest mirror and critical thinking partner, not a know-it-all. The project consists of markdown documentation defining prompts, skills, and behavioral analysis.

## Project Structure

- `PROMPT.md` - Initial prompt for starting conversations, defines core principles and vision
- `docs/research/self-analysis.md` - Critical behavioral analysis synthesized from 4 AI assistants
- `skills/_index.md` - Index for life-area skills (Wheel of Life methodology)
- `docs/architecture/` - System prompt and design decisions (to be created)

## Core Principles

When working on this project, maintain these design principles:

1. **Brutal honesty** - Critical and direct, no filters
2. **Scientific basis** - Evidence over myths
3. **No overfitting** - Simple and adaptable, not rigid
4. **Action over planning** - Detect and confront productive procrastination

## Key Context

The self-analysis document (`docs/research/self-analysis.md`) identifies patterns that LifeOS must actively counter:

- **Productive procrastination**: Planning as substitute for action
- **Complexity bias**: Creating systems/frameworks instead of simple solutions
- **Avoidance of discomfort**: Rationalizing delay of difficult tasks
- **Self-imposed overload**: Designing for infinite time/energy
